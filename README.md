# Impress My Guests - Multiplayer Home Designer Game

## Overview
Impress My Guests is a mobile multiplayer home designer game where 4-7 players create characters, design professional rooms, and compete in a voting-based competition. Similar to Dress to Impress, but focused on interior design and home decoration.

## Game Features

### Character Creation
- **15 Face Options** per gender
- **15 Hairstyle Options** per gender
- **Color Customization** for hair and clothing
- **Save Multiple Characters**

### Room Design
- **Furniture Library** with 50+ items
- **Color Palette Customization**
- **Theme Options**: Modern, Classic, Minimalist, Luxury, Casual
- **Professional Design Tools**
- **Real-time Preview**

### Multiplayer Lobby
- **4-7 Player Matches**
- **Game Code System** for friend invites
- **Matchmaking** for public games
- **Player Profiles** with statistics
- **Chat System** for in-game communication

### Voting & Scoring
- **1-5 Star Rating System**
- **Anonymous Voting** (optional)
- **Leaderboard Tracking**
- **Seasonal Rewards**
- **Score Multipliers** for consistency

### Monetization
- **In-Game Currency** (Coins & Gems)
- **Premium Character Skins**
- **Exclusive Furniture Packs**
- **Battle Pass System**
- **Ad-Free Version**
- **Sponsored Items** (IKEA, Fashion Brands, etc.)

## Technology Stack
- **Engine**: Unity 2022.3 LTS
- **Language**: C#
- **Backend**: Firebase (Realtime Database, Authentication, Cloud Functions)
- **Networking**: Firebase Realtime Database
- **UI Framework**: Unity UI (uGUI)
- **Version Control**: Git/GitHub

## Getting Started

### Prerequisites
- Unity 2022.3 LTS or later
- Firebase Console Account
- GitHub Account

### Installation
1. Clone the repository
2. Open in Unity
3. Configure Firebase credentials
4. Run in editor or build to device
