# Impress My Guests - Multiplayer Home Designer Game

A fun multiplayer game where players design their dream homes and vote on each other's designs!

## Features
- 👥 Multiplayer gameplay (4-7 players)
- 🏠 Room design with customizable furniture
- ⭐ Voting system with leaderboards
- 🛍️ In-game shop with monetization
- 🎨 Character customization
- 🔥 Firebase backend integration

## Quick Start
1. Clone the repository
2. Open in Unity
3. Follow SETUP.md for Firebase configuration
4. Play and enjoy!
